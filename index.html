17. Link States

HTML :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Link States Example</title>
    <style>
        body { font-family: Arial, sans-serif; }
        a { color: #4CAF50; text-decoration: none; }
        a:hover { text-decoration: underline; color: #3e8e41; }
        a:visited { color: #551A8B; }
        a:focus { outline: 2px dashed #4CAF50; }
        a:active { color: #FFFFFF; background-color: #4CAF50; }
    </style>
</head>
<body>

<h1>Link States</h1>

<p>Hover over this link: <a href="#">Hover Me!</a></p>
<p>This link changes color when visited: <a href="https://www.example.com">Visit Example.com</a></p>
<p>Click this link to see the focus and active state: <a href="#">Focus and Active Link</a></p>

</body>
</html>

10. How to Use Colspan and Rowspan in a Table

HTML :

<table border="1">
    <tr>
        <th colspan="2">Header 1 & 2</th>
        <th>Header 3</th>
    </tr>
    <tr>
        <td rowspan="2">Row 1, Cell 1 (spans 2 rows)</td>
        <td>Row 1, Cell 2</td>
        <td>Row 1, Cell 3</td>
    </tr>
    <tr>
        <td>Row 2, Cell 2</td>
        <td>Row 2, Cell 3</td>
    </tr>
    <tr>
        <td>Row 3, Cell 1</td>
        <td colspan="2">Row 3, Cells 2 & 3 (spans 2 columns)</td>
    </tr>
</table>

Click "Run Code" to see a table with colspan and rowspan attributes.

12. How to Use Box Shadow

HTML :

            
<style>
    body {
        font-family: Arial, sans-serif; 
        background-color: #f0f0f0; 
        padding: 20px; 
    }
    .container { 
        background-color: #f9f9f9; 
        padding: 20px; 
        border-radius: 8px; 
    }
    p { 
        box-shadow: 2px 2px 10px rgba(0, 0, 0, 0.1); 
        padding: 10px; 
    }
    fieldset { 
        box-shadow: 2px 2px 10px rgba(0, 0, 0, 0.2); 
        padding: 10px; 
    }
    img { 
        box-shadow: 5px 5px 15px rgba(0, 0, 0, 0.3); 
    }
</style>

<div class="container">
    <p>This is a paragraph with a box shadow.</p>
    <fieldset>
        <legend>Fieldset Title</legend>
        This fieldset has a box shadow.
    </fieldset>
    <img src="Your_img.jpg" alt="Sample Image">
</div>

15. Combined Semantic HTML Elements

HTML :

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Combined Semantic HTML Example</title>
    <style>
        body { font-family: Arial, sans-serif; }
        header { background-color: #4CAF50; color: white; padding: 10px; text-align: center; }
        section { margin: 20px 0; padding: 20px; background-color: #f0f0f0; border-radius: 8px; }
        article { padding: 10px; border: 1px solid #ccc; margin-bottom: 20px; }
        aside { background-color: #e0e0e0; padding: 10px; margin: 20px 0; }
        footer { text-align: center; padding: 10px; background-color: #4CAF50; color: white; }
    </style>
</head>
<body>

<header>
    <h1>My Blog</h1>
    <p>Welcome to my blog where I share insights and articles.</p>
</header>

<section>
    <article>
        <h2>Understanding Semantic HTML</h2>
        <p>Semantic HTML provides meaning to the web content, making it more accessible and easier to understand for both users and search engines.</p>
    </article>

    <aside>
        <h3>Related Topics</h3>
        <p>Check out these related topics to learn more about web development:</p>
        <ul>
            <li>HTML Basics</li>
            <li>CSS Fundamentals</li>
            <li>JavaScript Essentials</li>
        </ul>
    </aside>
</section>

<footer>
    <p>© 2024 My Blog. All rights reserved.</p>
</footer>

</body>
</html>
